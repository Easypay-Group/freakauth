FreakAuth library for CodeIgniter (http://www.codeigniter.com/):
FreakAuth Class Security handler that provides functionality
to handle login, logout, registration requests.
It also can verify the logged in status of a user and permissions.
The class requires the use of the Database and Encrypt CI libraries and the
URL, FreakAuth CI helper.  It also requires the use of the 3rd party DB_Session
library.  The FreakAuth library should be auto loaded in the core classes section
of the autoloader.
Passwords are encripted with md5 algorithm
